=plugin Bugtracker
metadata

This plugin sets the distribution's bugtracker URL and possibly the email
address as metadata.


=plugin FatPacker
file-munger

This plugin uses L<App::FatPacker> to pack your dependencies onto your
script file.


=plugin FindDirByRegex
file-finder

This plugin finds files in a directory and skips them if they match a
regular expression.

following file:
