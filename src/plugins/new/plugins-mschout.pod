=plugin ApacheTest
install prereqs template tests

This plugin will produce an L<ExtUtils::MakeMaker>-powered F<Makefile.PL>
with Apache::Test hooks for the distribution. If loaded, the
L<Manifest|Dist::Zilla::Plugin::Manifest> plugin should also be loaded, and
the L<MakeMaker|Dist::Zilla::Plugin::MakeMaker> plugin should not be
loaded.
At this time, this module is essentially a copy of the
L<MakeMaker|Dist::Zilla::Plugin::MakeMaker> plugin. Hopefully, over time,
the L<MakeMaker|Dist::Zilla::Plugin::MakeMaker> plugin will allow more
customization so that this module will not need to reimplement all of it.
